# 에러 레퍼런스

아래에 참조된 오류는 코드에서 발생할 때 아래 코드 중 하나에 해당하는 `lc_error_code` 속성을 갖습니다.

- [INVALID_CONCURRENT_GRAPH_UPDATE](./INVALID_CONCURRENT_GRAPH_UPDATE.md)
- [INVALID_GRAPH_NODE_RETURN_VALUE](./INVALID_GRAPH_NODE_RETURN_VALUE.md)
