# 애플리케이션에 대한 추적 활성화

애플리케이션에 대한 [추적](../concepts/tracing.md)을 활성화하려면 다음 환경 변수를 설정합니다:

```python
export LANGSMITH_TRACING=true
export LANGSMITH_API_KEY=<your-api-key>
```

자세한 내용은 [LangGraph로 추적하기](https://docs.smith.langchain.com/observability/how_to_guides/trace_with_langgraph)를 참조하세요.
