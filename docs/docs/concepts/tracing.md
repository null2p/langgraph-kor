# 추적

추적(Trace)은 애플리케이션이 입력에서 출력으로 이동하기 위해 수행하는 일련의 단계입니다. 이러한 개별 단계는 각각 run으로 표현됩니다. [LangSmith](https://smith.langchain.com/)를 사용하여 이러한 실행 단계를 시각화할 수 있습니다. 이를 사용하려면 [애플리케이션에 대한 추적을 활성화](../how-tos/enable-tracing.md)하세요. 이를 통해 다음을 수행할 수 있습니다:

- [로컬에서 실행 중인 애플리케이션 디버그](../cloud/how-tos/clone_traces_studio.md).
- [애플리케이션 성능 평가](../agents/evals.md).
- [애플리케이션 모니터링](https://docs.smith.langchain.com/observability/how_to_guides/dashboards).

시작하려면 [LangSmith](https://smith.langchain.com/)에서 무료 계정에 가입하세요.
